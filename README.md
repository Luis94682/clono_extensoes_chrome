Este repositório contém uma clonagem da página de extensões do Chrome, realizada como parte de um desafio pessoal/proposto por colegas.

O objetivo principal foi praticar e reforçar habilidades em HTML, CSS e JavaScript, e não há intenção de violar direitos autorais ou uso comercial do conteúdo original.

Todo o código foi desenvolvido para fins de estudo e aprendizado.
